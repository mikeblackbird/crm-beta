													-----Таблицы-----
		1. Clients
		Таблица представляет собой общий справочник клиентов.
		Причина создания: Хранилище данных о клиентах
			id[int] 		  – id Клиента
			firmname[varchar] - Имя фирмы
			facename[varchar] - Контактное лицо
			phone[varchar] 	  - Телефон
			email[varchar] 	  - Email
			comment[varchar]  - Комментарий к клиенту

		2. Divisions
		Таблица представляет собой список отделов.
		Причина создания: К каждому отделу привязываются работники. К каждому работнику привязываются задачи. 
			id[int]		   			– id Отдела
			divname[varchar]		- Название отдела
			divresponsible[varchar] - Начальник отдела
			phone[varchar]	   		- Телефон отдела
			Email[varchar] 	   		- Email отдела

		3. Workers
		Таблица представляет собой список работников. 
		Причина создания: Учет задач работников. Каждому работнику назначается задача.
			id[int]			   - id Работника
			name[varchar]	   - Имя работника
			perfomed[int]	   - Количество отработанных задач
			plan[int]	 	   - Месячный план
			idDivision[int]	   - ID.Divisions

		4. Orders
		Таблица представляет собой список заказов.
		Причина создания: Отслеживание и сохранение истории заказов
			id[int]			– ID Заказа
			name[varchar]	– Название заказа
			idClient[int]	– ID.Clients 		//Relation
			summary[int]	– Сумма заказа
			activity[int]	– Активность заказа. 0 - 1

		5. Services
		Таблица представляет собой список услуг
		Причина создания: Расчет стоимость заказов. Формирование проектов.
			id[int]			– ID Услуги
			name[varchar]	– Название услуги
			cost[int]		– Цена услуги
